# Space Fractions

## Overview
Space Fractions is a web-based, interactive learning tool designed to improve fraction-solving skills for sixth-grade students. The project follows a microservices architectural style and is deployed on a cloud-based infrastructure.

## Key Features
- Users can play the game, view their scores, and access help resources
- Admins can update the game questions
- The system is designed to be performant, secure, and scalable

## Deployment
The Space Fractions application is deployed on a cloud-based infrastructure. To deploy the application, follow these steps:

1. Ensure you have the necessary cloud infrastructure set up, including load balancing and autoscaling capabilities.
2. Deploy the following components:
   - GameServer
   - QuestionServer
   - UserClient
   - AdminClient
3. Configure the component interactions and secure the communication between services.
4. Set up monitoring and logging to ensure the system meets the performance, security, and availability requirements.
