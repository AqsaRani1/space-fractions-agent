# Space Fractions Project

## Overview

Space Fractions is a web-based, interactive learning tool designed to improve fraction-solving skills for sixth-grade students. It follows a microservices architectural style and is deployed on a cloud-based infrastructure using Node.js as the default language.

## Key Features

- Ensures the game responds quickly to user input (Performance)
- Implements encryption and secure authentication (Security)
- Implements load balancing and autoscaling (Scalability)

## Architecture

Space Fractions is built using a microservices architecture, with each service responsible for a specific functionality. The system is deployed on a cloud-based infrastructure for scalability and availability.

## Getting Started

To run the Space Fractions project, follow these steps:

1. Clone the repository
2. Install dependencies
3. Start the development server
4. Access the application in your web browser
